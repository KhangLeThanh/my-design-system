# Design System

## Overview

This Design System is a collection of reusable UI components built with React, TypeScript, and Storybook.
It ensures a consistent look and feel across applications while improving development speed and maintainability.

## Tech Stack

- React (UI components)

- TypeScript (type safety)

- Storybook (component documentation & playground)

- Vite (fast development & bundling)

- CSS

## Getting Started

### Install dependencies

```bash
npm start
```

### Run Storybook

```bash
npm run storybook

```

# my-design-system

Storybook will be available at:
http://localhost:6006
